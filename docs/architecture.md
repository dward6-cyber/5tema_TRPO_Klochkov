Описание архитектуры проекта

Компоненты технического стека

Flask (Python): Легковесный веб-микрофреймворк, примененный для построения модульной структуры приложения без избыточного базового кода.

SQLite: Реляционная встраиваемая база данных. Функционирует без развертывания выделенного сервера и сохраняет информацию в едином файле базы данных, что оптимально для демонстрационного прототипа.

SQLAlchemy (ORM): Инструмент объектно-реляционного отображения для абстракции от прямого написания SQL-запросов. Взаимосвязь таблиц Trip и Item настроена по принципу один ко многим с поддержкой каскадного удаления связанных объектов.

Bootstrap 5: Фронтенд-фреймворк, обеспечивающий адаптивный графический интерфейс и унифицированные стили веб-страниц.

Pytest: Среда автоматического тестирования, используемая для изоляции тестовой базы данных в оперативной памяти с целью проверки маршрутизации и валидации форм.
