# Руководство пользователя по планировщику отпуска

Как начать работу

1. Регистрация: При первом входе на сайт перейдите по ссылке "Зарегистрироваться", введите желаемый логин, пароль и нажмите кнопку "Создать аккаунт".
2. Авторизация: Введите свои данные на странице входа. После успешной авторизации в правом верхнем углу навигационной панели отобразится ваше уникальное имя пользователя.
3. Создание путешествия: На главной странице в левой панели "Новое путешествие" заполните поля: Название, место и даты. Нажмите кнопку "Создать чек-лист". Поездка появится в общем списке справа.

Управление вещами

1. Нажмите на название вашей поездки в списке - вы перейдете на детальную страницу чек-листа.
2. Внизу списка вещей введите название предмета (например, "Паспорт") и нажмите кнопку "Добавить".
3. Чтобы отметить вещь как собранную, нажмите зеленую кнопку "Собрано". Строка с вещью зачеркнется и обновится счетчик вещей.
4. Если вещь необходимо выложить, нажмите серую кнопку "Распаковать".

Использование шаблонизатора

1. Если в вашем аккаунте создано более одной поездки, внутри чек-листа станет доступен блок "Шаблонизатор".
2. Выберите из выпадающего списка прошлую поездку.
3. Нажмите кнопку "Скопировать список". Все вещи мгновенно перенесутся в текущее путешествие в статусе "не собрано".

Выход из системы
Для завершения сессии нажмите красную кнопку "Выйти" в правом верхнем углу шапки сайта. Доступ к вашим спискам закроется до следующего ввода пароля.
