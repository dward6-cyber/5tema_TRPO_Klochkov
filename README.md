Веб-сайт Планировщик отпуска (чек-листы вещей)

__________________________________________________________________________________________________


Данный проект представляет собой веб-сайт на базе микрофреймворка Flask, разработанное в рамках самостоятельной работы по дисциплине Технологии разработки ПО. Сайт помогает пользователям структурировать процесс подготовки к поездкам, отслеживать прогресс сборов и переиспользовать ранее созданные чек-листы.


__________________________________________________________________________________________________


Основной функционал:
Регистрация и аутентификация: Каждый пользователь работает в изолированном пространстве и видит исключительно свои данные.
Управление поездками: Создание, просмотр, редактирование параметров и удаление карточек путешествий.
Интерактивный чек-лист: Добавление вещей в поездку и переключение их статуса Собрано или Распаковать в один клик.
Поиск и фильтрация: Поиск необходимых поездок на главной странице по названию или локации.
Шаблонизатор: Возможность копирования списков вещей из одной прошлой поездки в текущую для экономии времени. Блок доступен только при наличии двух и более поездок в аккаунте.

Инструкция по установке и запуску

1. Подготовка
Склонируйте репозиторий и перейдите в папку с проектом. Для этого введите в терминал команду:
git clone https://github.com/dward6-cyber/5tema_TRPO_Klochkov.git
Далее перейдите в целевую директорию:
cd trip_plan

Установите необходимые зависимости:
pip install -r requirements.txt

2.Запуск сайта
Запустите сервер Flask:
python app.py
Приложение станет доступно в браузере по адресу http://127.0.0.1:5000/ или по проброшенному порту в виртуальной среде Codespaces.

3. Запуск автоматических тестов
Для проверки работоспособности маршрутов, авторизации, валидации форм и моделей данных выполните команду:
pytest
